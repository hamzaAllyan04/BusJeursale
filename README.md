# BusJeursalem

💻 Project Overview
This repository contains the source code for the "JR Bus" reservation system. The project provides a functional interface for managing travel bookings between Jerusalem and Ramallah.

🛠 Core Functionality
Static Web Architecture: Built using semantic HTML5 to ensure content structure.

Layout Design: CSS-based responsive layout handling different screen densities without external UI frameworks.

Client-side Logic: JavaScript implementation for handling user input, form validation, and dynamic content rendering based on selection (Payment methods & Seat allocation).

Communication Interface: Direct URI integration to WhatsApp for customer inquiries.

⚙️ Implementation Details
Frontend: Standard HTML/CSS/JS stack. No external libraries were used, keeping the codebase lightweight and performant.

Responsiveness: Managed via CSS media queries to ensure cross-device compatibility.

Deployment: Hosted via GitHub Pages, ensuring zero-configuration scalability.
